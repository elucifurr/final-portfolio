## Overview

ConilFix's website serves as a digital presence for a local repair shop in Conil de la Frontera. The static website provides essential information about the business's services, working hours, and contact details in a clear, accessible format. Built with Next.js and modern web technologies, the site ensures a fast, responsive experience for visitors seeking repair services. Visit the live site at [conilfix.com](https://conilfix.com).

## Key Features

- **Service Catalog**: Developed a comprehensive list of repair services with detailed descriptions.
- **Business Hours Display**: Implemented a clear presentation of working hours and availability.
- **Contact Information**: Created an easy-to-find contact section with phone numbers, email, and location.
- **Mobile-First Design**: Ensured optimal viewing experience across all device sizes.
- **Fast Loading**: Optimized performance for quick access to essential business information.

## Technologies Used

- **Next.js**: For creating a fast, static website with excellent SEO capabilities.
- **React**: To build an interactive and user-friendly interface.
- **TypeScript**: For enhanced code quality and maintainability.
- **TailwindCSS**: To implement a clean, responsive design system.
- **ShadCN**: For consistent and professional UI components.

## Challenges and Learnings

The main challenge was creating a simple yet effective design that would make it easy for customers to find the information they need quickly. This required careful consideration of information architecture and user experience. Another challenge was optimizing the site for local SEO to ensure the business would be easily discoverable in local search results.

## Outcome

The website has successfully established ConilFix's online presence, making it easier for local customers to find information about their repair services. The clean, professional design and fast loading times have contributed to improved customer engagement and accessibility of business information.
